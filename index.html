How this test works

View details

No data is collected by this app. Requests go directly from your browser to the selected public DNS-over-HTTPS resolvers.

Preparation

The app first prepares resolver connections so the measured run is less affected by first-request setup cost.

Measurement

Each resolver is checked against every hostname across multiple rounds using browser fetch and high-resolution timing.

Timeouts

Requests that fail or exceed the limit are counted as failed samples and scored with a 5000 ms penalty.

Variability

VPNs, mobile networks, browser extensions, routing, and resolver load can change results. Run more than once before changing DNS settings.

Results are browser DoH timings, not raw UDP DNS latency. Routing, VPN, CORS policy, and resolver load can change the ranking.
